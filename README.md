# Simple String Filter

```sh
ssf 0.1.0
Simple string filter

USAGE:
    ssf [OPTIONS]

OPTIONS:
    -d, --json-decode
    -e, --json-encode
    -f, --file <FILE>
        --force
    -h, --help               Print help information
    -o, --output <OUTPUT>
    -r, --raw
    -u, --lf
    -V, --version            Print version information
    -w, --crlf
```
